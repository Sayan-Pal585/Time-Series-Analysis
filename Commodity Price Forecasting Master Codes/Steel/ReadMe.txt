This compressed folder contains two folders, one for the python scripts and 
the other for the R scripts used in the building of the forecasting models.

R scripts contents:

1. UCM Univariate Model
2. GBM and regression based Model

Python scripts contents:

1. ARIMA (Steel and Rubber)
2. ARIMAX (Steel and Rubber)
3. Exponential Smoothing (Steel and Rubber)
   a.) Simple Exponential Smoothing
   b.) Holt's Linear Trend Method
   c.) Holt Winters Method
4. VARMA with production material
5. VARMA with similar metals
